ASUS

ASUS, é uma empresa multinacional de hardware e eletrônicos com sede no distrito de Beitou, Taipei, Taiwan. Essa já lançou bastante celulares baseados no Android da Google customização com a interface Zen UI.

Zen UI

ASUS ZenUI é uma interface de toque front-end desenvolvida pela ASUS com parceiros, apresentando uma interface de usuário totalmente touch. A versão atual é ZenUI 6.
